Frequently Asked Questions

1) What if no valid base can be formed?

If none of the 3-card base combinations sum to a multiple of 10, the hand has no valid arrangement under this ruleset.

2) Is A always treated as AS in input?

No. You can input plain A without suit. Direct Niu Dong Gu recognition requires explicit AS with J/Q/K as point cards.

3) Which is stronger: Pair or 10 Points?

In this implementation, multiplier is prioritized. Pair has multiplier 3 and 10 Points has multiplier 2, so Pair often ranks higher.

4) Why does the calculator sometimes choose lower-looking points?

Because ranking is not based on points alone. It compares multiplier first, then points, then kicker.

5) Should I manually test all 3/6 swaps?

You can, but it is error-prone in fast play. This calculator already checks every swap variant and returns the strongest legal result.
