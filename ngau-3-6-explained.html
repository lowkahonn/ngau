3/6 Swap in Ngau Explained

3/6 互換是很多人最容易忽略的規則。當你手上有 3 或 6，部分玩法會允許它們互換使用，令可行底牌組合和最終牌型發生變化。 This can turn a dead hand into a valid high-value hand.

How to Play Rules 3/6 Swap Example Rounds FAQ

What the Swap Means

In this calculator, every card that is 3 or 6 is evaluated in both states. If you have two swap-eligible cards, the tool checks all combinations of swapped and original states. 目的是確保不漏掉任何合法且更強的排法。

Why It Matters

  • It can create a valid base where no valid base existed before.
  • It can improve point-hand outcomes after base validity is satisfied.
  • It can change which pair should be assigned as point cards.

Practical Thinking Process

  1. List cards containing 3 and 6.
  2. Try base combinations with original values first.
  3. If no strong result, test swapped versions.
  4. Re-evaluate point pair after each valid base candidate.

Quick heuristic: if your current base total is close to a multiple of 10, one 3/6 swap can often unlock a legal split.

Important Detail

孖寶（pair）判定採用原始牌面（face rank）比較，不會因 3/6 互換而把不同牌面硬算成同一對。這樣可避免誤判。
